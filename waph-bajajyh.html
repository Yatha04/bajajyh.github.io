Course Description:

In this course on Web Application Development and Security, I delve into both the technical construction and security aspects of building web applications. I cover front-end technologies including HTML5, CSS, and JavaScript, as well as back-end development with PHP and MySQL. The course also explores web application vulnerabilities, introduces secure programming practices, and includes hands-on security exercises. Focused on practical skills, my curriculum aims to enhance my ability to design, implement, and secure robust web applications. Designed for individuals with intermediate programming skills, this course prepares me to address real-world web development challenges while implementing effective security measures to prevent and mitigate threats.

Hackathon Description:

In the hackathon on Cross-site Scripting Attacks and Defenses, I undertook tasks involving executing and defending against XSS attacks. I perform targeted attacks to display my name using alert functions across multiple complexity levels and deduce the underlying vulnerable code. In defense tasks, I enhance security through rigorous input validation and encoding practices, significantly bolstering my understanding of web security.

Lab 2 Description:

This lab spanned several sessions focused on basic web development and security. I crafted a simple HTML page integrated with JavaScript to manage dynamic content such as clocks and interactive forms. Further tasks involved Ajax, CSS, jQuery, and Web API integration to enhance interactivity and data handling on the page, improving my practical skills in web development and security.
